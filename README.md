# AVC-URCL Demo Page

Static demo page for the Antelope Valley College Office of Undergraduate Research & Creative Learning (URCL).

**Live demo:** https://avc-cs.github.io/AVC-URCL/

## Contents
- `index.html` — single-page demo with image carousel + URCL content
- `images/` — 20 photos shown in the Swiper.js carousel

## Stack
- Plain HTML / CSS (no framework, no build step)
- Swiper.js (loaded from CDN) for the image carousel
- Hosted on GitHub Pages
